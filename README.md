## 📖 Overview

A modern, AI-powered CSV import tool with drag-and-drop functionality, intelligent column mapping using Google Gemini AI, and comprehensive import history management.

## ✨ Features

### 🚀 Core Features

- **Drag & Drop Upload**: Intuitive file upload with visual feedback
- **AI-Powered Mapping**: Automatic column mapping using Google Gemini AI
- **Virtualized Tables**: Handle 10,000+ rows without performance issues
- **Import History**: Complete audit trail with detailed logs
- **Dark/Light Mode**: Full theme support with system preference detection
- **Real-time Validation**: Instant data validation with visual indicators
